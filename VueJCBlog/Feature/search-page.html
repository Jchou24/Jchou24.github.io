    Main Features

    • Reactive Full Text Search

      • 在 Search 頁面中，任何的搜尋操作都會有及時的搜尋結果反饋(Reactive)

      • 目前主要支援的語言有 Chinese, English

      • 全文搜尋

      • Query Auto Complete

        • 搜尋時輸入關鍵字的部分，將出現可能的關鍵字清單

      • Excerpt

        • 顯示內文摘要

      • Key Word Cloud

    • State Remember

      • 離開搜尋頁面後，當前搜尋的結果以及條件將會被保留

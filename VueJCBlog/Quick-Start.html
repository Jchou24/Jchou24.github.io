• Download

    • Node.js

      tested npm version: 6.9.0

    • Python2.7

      由於全文搜尋在建置過程中使用到了 nodejieba 套件

      因此安裝相依套件nodejieba時，會需要使用到Python

    • source code

      於下載後，資料夾結構應如下

      .
├── docs
│   └── README.md
│
├── lib/
│   ├── ...
│   ...
│
├── package.json
└── package-lock.json

  • 安裝相依套件

    • Basic Step

      cd path/to/VueJCBlog/folder
npm ci # or
npm run install

    • Detail

      由於使用到的相依套件的關係，若無法按照Basic Step來進行安裝

      更詳盡的安裝內容請參考 Installation

  • Start to write article

    • Hello VueBlog

      1. 於docs/底下建立hello-vuejcblog.md

        .
├── docs
│   ├── README.md
│   └── hello-vuejcblog.md(Add this file)
│
├── lib/
│   ├── ...
│   ...
│
├── package.json
└── package-lock.json

      2. 貼上以下內容

        ---
title: Hello VueJCBlog
---

# Hi

* This is my first article.

    • Detail

      • 更多支持的Markdown功能請參考 Write Article with Markdown

    • Run dev

      1. 於command line執行

        npm run dev

      2. 於browser開啟 http://localhost:8080/hello-VueJCBlog.html

  • Deploy

    • Build

      於command line執行

      npm run build

    • Upload

      請參考 Vuepress官方說明

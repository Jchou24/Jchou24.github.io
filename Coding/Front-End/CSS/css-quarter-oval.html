Case 1

令一邊border-radius是100%畫圓，透過長寬去變形成橢圓

.quarter-oval-1-1{
    background: #a1b7bd;
    height: 100px;
    width: 100px;
    border-top-right-radius: 100%;
}

.quarter-oval-1-2{
    background: #a1b7bd;
    height: 100px;
    width: 200px;
    border-top-right-radius: 100%;
}

Case 2

完全靠border-radius來調整

.quarter-oval-2-1{
    background: #a1b7bd;
    height: 100px;
    width: 200px;
    border-radius: 0 100px 0 0 / 0 100px 0 0;
}

.quarter-oval-2-2{
    background: #a1b7bd;
    height: 100px;
    width: 200px;
    border-radius: 0 200px 0 0 / 0 100px 0 0;
}

Case 3

靠背景漸層來做到

.quarter-oval-3{
    height: 100px;
    width: 200px;
    background-image: radial-gradient(ellipse at bottom left, #a1b7bd 60%, transparent 70%);
}
